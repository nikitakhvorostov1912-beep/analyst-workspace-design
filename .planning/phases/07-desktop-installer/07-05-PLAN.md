    - [ ] Закрыть окно крестиком

    - [ ] Нет orphan процессов в фоне

    - [ ] Двойной клик ярлыка ещё раз → работает (random freeport, нет конфликта портов)
    - [ ] SQLite в %APPDATA%/1С Аналитик/app.db создан и содержит данные из первого запуска
