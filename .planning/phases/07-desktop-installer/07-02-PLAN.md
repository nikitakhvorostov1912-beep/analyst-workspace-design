<objective>
Упаковать FastAPI backend в один исполняемый файл `backend.exe` (~50 MB) через PyInstaller `--onefile`. Файл должен включать Python runtime + uvicorn + все зависимости (aiosqlite, sse-starlette, pydantic-settings, httpx) и запускаться без установленного Python в системе.

Purpose: Закрыть DIST-02 — без bundled backend.exe Electron main process не может spawn'ить backend на машине аналитика, у которого нет Python.

Output: `backend/dist/backend.exe` (артефакт build-а), `backend/app/main_cli.py`, `backend/build.spec`.
</objective>

  <files>backend/app/main_cli.py</files>
  <action>
    Создать файл backend/app/main_cli.py — точка входа для bundled .exe. Использует argparse для приёма --port (int, default 8010) и --host (str, default '127.0.0.1'). Импортирует `app` из `app.main` (НЕ создавать новый app внутри — используем уже сконструированный singleton из main.py:72, чтобы все routes и lifespan уже были подключены). Запускает `uvicorn.run(app, host=args.host, port=args.port, log_level='warning')`.

    Внутри файла: `def main()` парсит аргументы, `if __name__ == '__main__': main()`.

    Никаких дополнительных middleware/routes здесь не добавлять — это просто programmatic wrapper для uvicorn CLI. Все настройки берутся из app.config.Settings (через env DATABASE_URL, BACKEND_ALLOWED_ORIGINS, переданные Electron main процессом).

    Karpathy: файл ≤30 строк, никаких абстракций, никакого error handling вокруг uvicorn.run (uvicorn сам пишет в stderr при ошибке — Electron parent process прочитает).

  <name>Task 2: Создать build.spec и добавить pyinstaller в pyproject.toml</name>
  <files>backend/build.spec, backend/pyproject.toml</files>
  <action>
    1. Дополнить backend/pyproject.toml — добавить в `[project.optional-dependencies]` новую группу `build = ["pyinstaller>=6.10"]`. Существующие группы (dev) НЕ трогать.

    2. Создать backend/build.spec (PyInstaller spec file). Это python-файл, выполняемый PyInstaller'ом. Структура — см. 07-CONTEXT.md секция Plan 7.2:
       - `a = Analysis(['app/main_cli.py'], pathex=['.'], datas=[('app', 'app')], hiddenimports=[...], excludes=[...])`
       - hiddenimports: ['uvicorn.logging', 'uvicorn.protocols', 'uvicorn.protocols.http', 'uvicorn.protocols.http.auto', 'uvicorn.protocols.websockets', 'uvicorn.protocols.websockets.auto', 'uvicorn.lifespan', 'uvicorn.lifespan.on', 'aiosqlite', 'sse_starlette', 'pydantic', 'pydantic_settings']
       - excludes: ['tkinter', 'matplotlib', 'numpy', 'pandas'] — уменьшение размера
       - `pyz = PYZ(a.pure)`
       - `exe = EXE(pyz, a.scripts, a.binaries, a.datas, name='backend', console=False, onefile=True, icon='../desktop/icon.ico' if exists else None)` — console=False чтобы не было чёрного cmd окна при spawn'е; icon вычисляется через `os.path.exists` чтобы не падать когда desktop/icon.ico ещё не создан (Plan 7.4 его создаст)

    Karpathy: spec ≤50 строк, без дополнительных datas кроме 'app' (миграции лежат в app/storage/, попадут автоматически).
  </action>
