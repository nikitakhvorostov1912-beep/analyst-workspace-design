`_dispatch_query_card(args, result)` (новая функция):
1. Парсит data из result (как _build_table_card)
2. Если rows=1 AND есть ≥1 numeric column → MetricCard (без sparkline)
3. Если rows ≤ 100 AND есть колонка matching `r'^(Период|Дата|Месяц|Date|Period)$'` AND ≥1 numeric column → MetricCard со sparkline. Value = последнее значение numeric column, label = name numeric column. Delta = (last - first) / first * 100.
4. Иначе → _build_table_card (текущее поведение)
